/API/Login POST:
    Authorization: none
    required form fields:
    {
    	login: username(string)
        password: password(string)
    }
    optional arguments: none

    response: on success simply 200, on failure 400 error with error messsage.




/API/Account POST:
    required form fields:
    {
        username: string,
        password: string,
        name: string,
        surname: string,
        day: int,
        month: int,
        year: int,
        email: string,
        address: {
            postal_code: string,
            house_number: int, 
            city: string, 
            country: string
        }
    }

    response: on success simply 200, on failure 400 error with error messsage.
